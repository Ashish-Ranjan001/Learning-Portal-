CSV Format Instructions

Your CSV file should follow this format:

  • First column: Question text
  • Columns 2-5: Answer options
  • Column 6: Correct answer number (1-4)

Example:

Question,Option1,Option2,Option3,Option4,CorrectAnswer
"What is Angular?","Framework","Language","Library","Tool",1
"Which version is latest?","17","18","19","20",3
